## 1. 問題陳述

### 1.1 目標使用者

| 族群 | 規模 | 痛點 |
|---|---|---|
| YouTube 創作者 | ~5 萬 | 想跟上熱門話題但要花時間搜尋、無法快速定位 |
| 行銷公司 | ~3,000 | 客戶要產出跟上時事的內容，需快速找到相關熱門影片 |
| 內容研究者 | ~5,000 | 想分析特定主題的熱門影片趨勢 |
| 一般使用者 | 不限 | 想找到特定主題的優質影片但 YouTube 搜尋結果太雜 |

### 1.2 為什麼不做替代方案

- **YouTube 搜尋**：結果太多、需要時間篩選
- **商用工具（Tubular Labs / Vidooly）**：月費 100-1000 USD，過於昂貴
- **現有開源工具**：資料來源單一、UI 不友善
- **我們的解法**：多維度排序（觀看數/按讚數/留言數/發布時間/長度）+ 主題分類 + 匯出 CSV，純前端

---

## 2. 解決方案

### 2.1 核心價值主張

> 「輸入主題，30 秒拿到 100 支熱門影片 + 多維度排序。」

### 2.2 使用者流程

1. 輸入主題關鍵字（例如「AI 教學」「減重」）
2. 選擇時間範圍（24 小時/7 天/30 天/全部）
3. 系統抓取 YouTube Data API 結果
4. 多維度排序（觀看/讚/留言/時長/發布日）
5. 一鍵匯出 CSV / 加入我的最愛

---

## 3. 功能清單

### 3.1 MVP（必做）

- [ ] 主題關鍵字搜尋
- [ ] 時間範圍篩選（24h/7d/30d/all）
- [ ] 5 維度排序（觀看/讚/留言/時長/發布日）
- [ ] 影片卡片（含縮圖/標題/頻道/數據）
- [ ] CSV 匯出
- [ ] 我的最愛清單
- [ ] 主題分類預載（10 種熱門主題）

### 3.2 v2（加值）

- [ ] 趨勢圖表（每日熱門排行變化）
- [ ] 同類型影片比較
- [ ] 自動字幕下載
- [ ] 多關鍵字組合搜尋

### 3.3 明確不做

- 影片下載（純資訊工具）
- 自動評論/回覆（不做 YT 互動）
- 商業版權影片標記
- 與其他平台整合（純 YouTube）

| 前端 | Next.js + TypeScript | SSR 友善 |
